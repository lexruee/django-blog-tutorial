# Simple Django Blog Example

## Create virtualenv

```
pipenv --three
```

## Enter the virtualenv

```
pipenv shell
```

## Install Django

```
pip install Django
```


## Run migrations

```
./manage.py makemigrations 
./manage.py migrate 
```

## Run tests

```
./manage.py tests
```

## Run Blog

```
./manage.py runserver 8000
```
